# User Stories

## Specific Device Capability Page

1. As a standard user of this tool, I want to be able to query for a specific device, and see all global capabilities that that device has including all features, bands, and combos to answer customer requests, or select specific devices for testing

1. As a standard user of this tool, I want to be able to query for a specific device, and see all capabilities that that device has including all features, bands, and combos for a unique telecom provider (Telus, Rogers, etc) to answer customer requests, or select specific devices for testing


## Capability Querying Page

1.  As a standard user of this tool, I want to be able to query for a list of devices that support a given feature (this is across all telecom providers. I.e. if a device is capable of VoLTE, but VolTE on that device is only supported by one telecom provider, it should still be shown) to answer customer requests, or select specific devices for testing

1. As a standard user of this tool, I want to be able to query for a list of devices that support a given feature for a specific telecom provider to answer customer requests, or select specific devices for testing

1. As a standard user of this tool, I want to be able to query for a list of devices that support a given band (either GSM, HSPA, LTE, NR) (This should query the global capability for all devices and return devices and their associated softwares that support this band) to answer customer requests, or select specific devices for testing

1. As a standard user of this tool, I want to be able to query for a list of devices that support a given band (either GSM, HSPA, LTE, NR) in a specific telecome providers network(This should query the telecom capability for all devices and return devices and their associated softwares that support this band) to answer customer requests, or select specific devices for testing

1. As a standard user of this tool, I want to be able to query for a list of devices that support a given CA combo (either LTE, ENDC, or NR CA) (This should query the global capability for all devices and return devices and their associated softwares that support this LTE combo) to answer customer requests, or select specific devices for testing

1. As a standard user of this tool, I want to be able to query for a list of devices that support a given CA combo (either LTE, ENDC, or NR CA) in a specific telecom providers network. (This should query the telecom capability for devices in the network and return devices and their associated softwares that support this combo) to answer customer requests, or select specific devices for testing

1. As a standard user of this tool, in all queries I construct on this page, I want to be abe to specify which fields are returned related to the device
